# 计算机专业的第一课

> [!tip] 你应该如何阅读该教程？
> 
> Case1：我有编写程序的经验。
> 
> &emsp;那此教程不适合你。
> 
> Case2：我想最快推进进度。
> 
> &emsp;那你可以跳过各种Q&A（问题和回答）板块、可选任务板块。这些板块是为了提升计算机使用能力设计的。但是你会失去很多学习知识的机会。
> 
> Case3：我连使用电脑都不是很熟悉。
> 
> &emsp;那此教程就是为你打造的。请仔细阅读该教程。

---
